### 🎮 游戏设计文档 (GDD - 完整版)

**游戏名称：** 《营造录：寒门赴考》 (The Builder's Journey)
**视觉风格：** 2D 等距视角 (Isometric)，国风水墨与青绿山水结合。
**核心循环：** 大地图探索（旋转道路接通） ➡️ 进入建筑关卡（解密/轻动作） ➡️ 获得关键构件 ➡️ 解锁新区域。

#### 一、 主剧情与地图交互系统设计
*   **故事主线：** 贫苦书生从小立志科举。游戏地图就是一张从“江南水乡”一路延伸至“京城紫禁”的立体画卷。
*   **大地图交互（魔方道路系统）：**
    *   整个大地图由悬浮的 2D 等距地块拼接而成。书生走到两个区域的交界处时，前方是没有路的。
    *   玩家需要进入关卡，通关后获得“鲁班令”。
    *   消耗“鲁班令”，玩家可以像扭转魔方一样，顺时针/逆时针旋转交界处的大型 3D/2.5D 建筑地块（如一截断裂的廊桥、一段错位的城墙）。当道路纹理首尾相连时，金光闪烁，书生自动走向下一站。

#### 二、 关卡设计（玩法与建筑主题深度融合）

我们选取你清单中最具可行性的几种玩法，进行“建筑学”爆改：

**第一章：乡绅大院（对应建筑：徽派民居）**
*   **场景表现：** 典型的徽派建筑风格，“粉墙黛瓦马头墙”。等距视角下，向玩家立体展示一个带有深邃天井的青砖大院。游戏过程中，雨水会顺着四面的内向坡屋顶汇聚到中央的天井中，不仅可以作为解密的视觉提示，更生动呈现了中国传统民居中“四水归堂”（寓意肥水不流外人田，聚气聚财）的建筑与风水理念。
*   **剧情设定：** 书生为筹集进京赶考的盘缠，发挥所学，在富商乡绅家接下了修缮祖宅天井的活计。
*   **核心玩法：【古代图纸推演】+【三视图建筑拼图】**
    *   **图纸解析（目标展示）：** 降低硬核的物理操作门槛，改为更大众的“逻辑拼图”玩法。修缮开始时，系统会向玩家展示一张修缮完毕后的**“建筑实物参考图”**（例如一角带有雕花雀替和瓦当的精美飞檐）。
    *   **收集零件：** 玩家在天井院落的各个房间内探索，通过简单的点击寻物，在杂物堆、水缸和院落角落中收集散落的各类建筑构件（如：瓦当、雀替、木梁）。
    *   **三视图图纸拼装（核心操作）：** 收集完毕后，进入“营造案台”界面。桌面上铺开一张古风图纸，上面画开了该建筑结构的**正视图、俯视图、侧视图**三个线框图。
        *   玩家需要根据观察之前给出的“实物图”，利用空间想象力，将对应的零件碎片拖拽到这三张二维视图的轮廓阴影中。
        *   *玩法表现：* 例如“兽纹瓦当”，虽然实物是个立体圆筒，但在“正视图”里是个圆圈，在“俯视图”里却是一块带弧度的长条。玩家无需懂建筑承重学，只需像玩“七巧板”或“图形对对碰”一样，把相应的二维切面图拼进框架里即可。
    *   **文化反馈：** 当三视图的图纸全部拼凑正确无误后，图纸上的二维构件会产生“起凸”特效，以水墨化作立体的3D形体，自动飞去填补天井的缺损处；同时侧边栏弹出中国古代图样绘制（如清代著名的“样式雷”图档）的科普卡片，轻松科普建筑文化。
*   **过关奖励：** 乡绅惊叹于书生展现出的空间几何推演与营造设计技艺，慨然赠送丰厚的盘缠，书生拿到通关的“鲁班令”。

**第二章：风雨险途（对应建筑：桥梁）**
*   **剧情：** 离开家乡，路遇险恶峡谷，一座古老的木拱廊桥被山贼占据。
*   **融合玩法：【Q版泡泡堂】+【塔防】**
    *   **玩法表现：** 采用 2D 俯视角。山贼（劫匪）从桥的一端不断涌来，向书生扔石头。书生没有武器，但可以利用廊桥的建筑机关！
    *   玩家像放泡泡堂的炸弹一样，在桥面上放置“机关陷阱”（如：触动榫卯让桥面暂时塌陷、放下廊屋的挡风板阻挡石头、滚落圆木）。利用桥梁的结构特性，将山贼击退或困住。

**第三章：奇门遁甲（对应建筑：官府/城关）**
*   **剧情：** 抵达京城外的最后一道关卡，守城官吏故意刁难，要求书生破解城门阵法。
*   **融合玩法：【森林冰火人变体】（双线控制）**
    *   **玩法表现：** 巨大的城门由复杂的齿轮和水力驱动。玩家需要同时控制书生和一只“机关木鸟”（或者分饰两角）。书生在下方踩踏机关改变水流方向，木鸟在上方衔取钥匙开启锁孔。两人必须配合默契，利用建筑内部的高低差和机关联动，打开城门。

**第四章：金榜题名（对应建筑：皇宫）**
*   **剧情：** 步入紫禁城，太和殿上，皇帝亲自出题考核。
*   **融合玩法：【硬核空间解密】**
    *   **玩法表现：** 皇帝拿出一套极其精巧的“样式雷烫样”（微缩皇家建筑模型）。玩家需要在华丽的宫廷背景下，完成最后的高难度榫卯拼装。拼装成功，模型散发光芒，书生被钦点为状元。

#### 三、 营造图鉴系统（文化与知识收集）

为了进一步强化“中国古代建筑成就”的文化科普属性，游戏内将内置一套丰富的**《营造图鉴》**系统。这不仅是玩家的游戏成就展示柜，更是可以反复查阅的古建筑知识库。

*   **图鉴触发机制（如何获取知识卡片）：**
    *   **新景解锁：** 每当主角抵达一种全新的建筑类型地块（如：首次进入徽派民居、首次踏上木拱廊桥、首次遥望皇家宫殿）时，会在屏幕中心弹出包含该建筑全景和历史背景的“大图鉴卡”。
    *   **构件修复：** 在具体的关卡中，当玩家成功解密、拼装或发现一个特定的建筑构件（如：雕花雀替、穿插枋、鸱吻、斗拱）时，会在侧边栏以水墨特效收集到一张对应的“构件小图鉴卡”。
    *   **彩蛋彩蛋：** 场景中隐藏着一些特定的可以互动的陈设（如：古井、石库门、影壁），点击探索也可获得冷知识卡片。

*   **图鉴卡片内容设计（UI表现）：**
    *   每张卡片采用竹简或泛黄宣纸的底纹风格。
    *   **正面**显示该构件/建筑的精美手绘线稿（支持 3D 拖拽旋转查看结构）。
    *   **背面**记录该部位的：1. 准确读音与名词解释。2. 在古建筑力学中所承担的实际功能（如：承重、防雨、装饰、等级象征）。3. 相关的工程图纸案例（如引用《营造法式》或“样式雷图档”的原文片段）。

*   **完成度奖励：**
    *   收集一定比例的图鉴，可以解锁书生在科举过程中的特殊称号或服饰外观。
    *   达成 100% 收集（一本完整的《营造录》），可以在通关后解锁特殊的“大好河山”全屏画卷CG，极大提升玩家的成就感和文化自豪感。
