# 营造录：寒门赴考 (The Builder's Journey)

> 第 X 届中国大学生计算机设计大赛 - 数媒游戏与交互设计 参赛作品

## 📜 项目背景与主旨
《营造录：寒门赴考》是一款以中国古代建筑文化为核心的 2D 等距视角冒险解密游戏。
游戏以一名寒门书生的赶考之路为主轴，串联起**民居、桥梁、官府、皇宫**四大建筑类型。我们抛弃了枯燥的说教，将古代建筑学原理（如榫卯、受力结构、机关术）巧妙融入到大地图的“魔方道路旋转”以及各具特色的关卡玩法中，旨在以数字化、交互式的游戏语言，向中华优秀传统文化致敬。

## 🌟 核心特色机制
1. **等距视角与魔方矩阵：** 大地图采用 Isometric 视角。玩家通过通关获得能量，旋转并重组错位的建筑地块，打通赴考之路，极具空间拼图的爽快感。
2. **多维度的建筑解密：**
   - **民居篇：** 融合滑轮抓取的修缮玩法，体验“四水归堂”的结构美学。
   - **桥梁篇：** 结合轻塔防机制的廊桥保卫战，利用建筑机关巧退山贼。
   - **皇宫篇：** 沉浸式的“烫样”（微缩模型）榫卯拼装，挑战空间逻辑极限。
3. **国风视听体验：** 采用精致的手绘 2D 贴图与序列帧动画，配合古风民乐，营造高度沉浸的东方美学氛围。

## 🛠️ 技术架构
* **开发引擎：** Godot 4.x / Unity 2022 LTS (请根据最终选择填写)
* **编程语言：** GDScript / C#
* **核心系统：**
  - `TileMapSystem`：基于二维数组与邻接矩阵的道路连通性校验算法。
  - `InteractableMechanism`：通用机关状态机，用于控制各关卡中桥梁塌陷、齿轮转动等状态。
  - `DialogSystem`：轻量级 JSON 驱动的剧情对话系统。
